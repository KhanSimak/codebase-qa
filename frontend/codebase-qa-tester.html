First time? One backend change is needed. FastAPI blocks cross-origin requests by default, and this page (opened as a local file) counts as a different origin than your API. Add this to app/main.py right after app = FastAPI(...), then restart the server:
from fastapi.middleware.cors import CORSMiddleware
app.add_middleware(CORSMiddleware, allow_origins=["*"], allow_methods=["*"], allow_headers=["*"])
Once requests below succeed, this note won't matter — feel free to ignore it if CORS is already enabled.
